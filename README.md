# Pebble

A minimal and clean Big Picture Mode theme featuring a modern rounded aesthetic, an immersive home screen layout, and a minimal distraction-free game view.

![Hero Image](placeholder-hero.png)

## Features

*   **Modern Aesthetic**: Creates a clean, uncluttered interface to focus on what matters: your games.
*   **Distraction-Free Game View**: A minimalist library view with a focus on displaying game art rather than clutter.
*   **Highly Customizable**: Fine-tune corner radii, transparency, focus outlines, and more to match your preferences.

## Configuration Options

Pebble offers extensive customization options directly through the Millennium theme settings:

### Interface
*   **Corner Radius**: Adjust the roundness of UI elements (0px - 20px).
*   **Focus Outline**: Choose between Rounded, Square, or Hidden focus rings.
*   **Transparency**: Control desktop top bar transparency (0% - 90%).
*   **Force Transparency**: Optionally force transparency on the top bar.
*   **Styling**: Apply rounded aesthetics specifically to Big Picture Mode and even Desktop Mode.

### Home
*   **Friends Widget**: Toggle the visibility of the friends bar on the home screen.
*   **Home Filter**: Enable or disable the dark filter on the home screen.
*   **Vertical Offset**: Adjust the vertical position of the hero image (0px - 50px).

### Library
*   **Action Bar Style**: Select from Transparent, Blur, or Connected styles for the game action bar.
*   **Artwork Alignment**: Customize how game header artwork is aligned: Zoom-Center, Zoom-Left, Zoom-Right, Contain, or Stretch.
*   **Cloud Sync Icon**: Show or hide the Steam Cloud sync icon.
*   **Header Layout**: Choose between Standard or Full Height header layouts.
*   **Logo Anchoring**: Pin the game logo to the bottom of the header or let it float freely.
*   **Clean Game Launch**: Enable a minimalist game launch screen without gamepad animation.

## Installation

1.  Download the latest release of Pebble.
2.  Install [Millennium](https://github.com/SteamGridDB/millennium-client) if you haven't already.
3.  Place the `Pebble` folder into your `<Steam>/steamui/skins` directory.
4.  Select "Pebble" from the Millennium settings in Steam.
